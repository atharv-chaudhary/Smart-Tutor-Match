## Overview

Smart Tutor Match is a front-end prototype of a tutor-matching platform. The goal of this project is to help students find tutors based on their 

academic requirements such as subject, class level, and preferred mode of learning.



## Problem Faced

Many students struggle to find tutors who match their exact needs. This problem is more common in smaller towns where access to quality 

academic support is limited. This project explores how a simple web-based system can help solve this issue.



## How the Website Works

1. The student enters subject, class, and learning mode on the homepage.

2. On submitting the form, the data is passed to a results page.

3. The results page displays sample tutor profiles that match the student's requirements.

4. Tutor profiles are displayed in a grid-based card layout for better readability and user experience.



## Technologies Used

- HTML: Used to structure the web pages.

- CSS: Used for layout, design, and card-based sections.

- JavaScript: Used to transfer user input and dynamically display tutor information.

learned how to break a real-world problem into smaller technical components.
